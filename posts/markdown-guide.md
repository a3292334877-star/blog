# Markdown 写作指南 📝

Markdown 是一种轻量级标记语言，用纯文本格式就能写出结构清晰的文档。这篇笔记整理了常用语法和写作技巧。

<!-- more -->

## 基础语法

### 标题

用 `#` 表示标题层级，1~6 级：

```md
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式

| 效果 | 语法 |
|------|------|
| **粗体** | `**粗体**` |
| *斜体* | `*斜体*` |
| ~~删除线~~ | `~~删除线~~` |
| `行内代码` | `` `行内代码` `` |

### 链接与图片

```md
[链接文字](https://example.com)
![图片描述](https://example.com/image.png)
```

### 列表

无序列表用 `-` 或 `*`，有序列表用数字：

```md
- 项目一
- 项目二
  - 嵌套子项

1. 第一步
2. 第二步
```

### 引用

```md
> 这是一段引用文字
> 可以多行
```

### 代码块

用三个反引号包裹，指定语言可以获得语法高亮：

````md
```python
def hello():
    print("Hello, Markdown!")
```
````

效果：

```python
def hello():
    print("Hello, Markdown!")
```

### 分隔线

三个或更多的 `---`、`***`、`___`：

---

### 表格

```md
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
| D   | E   | F   |
```

## 扩展语法

不同平台支持的扩展可能不同，以下是常用扩展：

### 任务列表

```md
- [x] 已完成
- [ ] 待办事项
```

### 脚注

```md
这是一段带脚注的文字[^1]

[^1]: 这是脚注内容
```

### 数学公式

VitePress 配合 `markdown-it-mathjax3` 支持 LaTeX 数学公式：

行内公式：`$E = mc^2$` → $E = mc^2$

块级公式：

```md
$$
\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n
$$
```

### Emoji

```md
:smile: :rocket: :heart:
```

## Frontmatter 配置

在 `.md` 文件顶部用 `---` 包裹 YAML 配置：

```yaml
---
title: 文章标题
date: 2026-06-01
tags: [教程, Markdown]
cover: https://example.com/cover.jpg
---
```

本博客支持的 frontmatter 字段：

| 字段 | 说明 | 必填 |
|------|------|:--:|
| `title` | 文章标题 | ✅ |
| `date` | 发布日期 | ✅ |
| `tags` | 标签列表 | ✅ |
| `cover` | 封面图 URL | ❌ |

## 写作建议

> 好的排版让阅读更愉悦

- **善用标题层级**，让文章结构清晰
- **段落之间空一行**，避免文字拥挤
- **代码块要指定语言**，获得语法高亮
- **适当使用列表和表格**，提高信息密度
- **配图合理**，大段文字中加入图片缓解阅读疲劳
